# CyberHub — Advanced Digital Safety Portal

A unified, high-performance cybersecurity platform built with React, Vite, and Framer Motion. This project combines real-time threat intelligence with secure incident reporting.

## Key Modules
- **Command Center (Home)**: Real-time status overview and global stats.
- **Reporting Protocol**: Anonymized, multi-step incident reporting system.
- **Intel Terminal**: Live CVE lookup from NVD and dynamic threat news feed.
- **Cyber Assistant**: Integrated AI chatbot for immediate security guidance.

## Tech Stack
- **Frontend**: React 18, Vite, Framer Motion (Animations), Lucide (Icons).
- **Styles**: Custom Glassmorphism design system in Vanilla CSS.
- **Data**: Axios for NVD API integration.

## Getting Started
1. `npm install`
2. `npm run dev`

## Deployment
Build for production:
`npm run build`
